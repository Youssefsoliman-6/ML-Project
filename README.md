# 💻 Laptop Price Prediction – ML Project

A machine learning project that predicts laptop prices (in Euro) using regression models trained on real laptop specifications data.

## 📌 Project Overview

This project explores and compares multiple regression algorithms to predict laptop prices based on features like RAM, Weight, and Brand. It includes full data preprocessing, model training, evaluation, and hyperparameter tuning.

## 📂 Files

| File | Description |
|------|-------------|
| `ML_Project.ipynb` | Main Jupyter notebook with full ML pipeline |
| `ML_Project_Report.docx` | Project report with findings and analysis |

## ⚙️ Models Used

- Linear Regression
- Polynomial Regression (degree=2)
- Ridge Regression
- Lasso Regression
- Decision Tree Regressor

## 🛠️ Tech Stack

- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib & Seaborn

## 📊 Evaluation Metrics

- RMSE (Root Mean Squared Error)
- MAE (Mean Absolute Error)
- R² Score
- K-Fold Cross Validation (5 splits)

## 🚀 How to Run

1. Clone the repository
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Open `ML_Project.ipynb` in Jupyter Notebook
4. Run all cells
